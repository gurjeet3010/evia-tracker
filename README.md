# 🌸 Evia — Smart Cycle & Wellness Tracker

**Evia** is a modern, user-centric menstrual health tracking web app designed to help users understand their cycle, log symptoms, and gain meaningful insights through a clean and calming interface.

🔗 **Live Demo:** https://evia-tracker.lovable.app

## 📱 Install as a PWA (Progressive Web App)

Evia can be installed on your device like a native app for quick access and a better experience.

### 💻 On Desktop (Chrome / Edge)

1. Open the app: https://evia-tracker.lovable.app
2. Look for the **Install icon (➕)** in the address bar
3. Click **"Install Evia"**
4. The app will be added to your desktop/home screen

---

### 📱 On Android (Chrome)

1. Open the app in Chrome
2. Tap the **three-dot menu (⋮)**
3. Select **"Add to Home screen"**
4. Confirm installation

---

### 🚀 Why Install?

* Faster access (no need to open browser every time)
* App-like experience (full screen, smoother UI)
* Works like a native wellness app

---

> ⚠️ Note: PWA installation works only if the app supports a valid manifest and HTTPS (already enabled in most hosted apps like this one).

## ✨ Overview

Evia focuses on making menstrual tracking:

* Simple → no overwhelming inputs
* Insightful → meaningful patterns over time
* Comfortable → soft UI, stigma-free experience

It blends **intuitive design + data tracking** to create a supportive digital wellness companion.
